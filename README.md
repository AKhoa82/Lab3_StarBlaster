## 📌 Mô tả
Đây là bài thực hành Lab 03 môn **Game 2D Development with Unity**.  
Mục tiêu của bài lab là:
- Làm quen với dự án game thực tế
- Hiểu cấu trúc project Unity
- Phân tích code và gameplay
- Thực hành chỉnh sửa game

---

## 🛠️ Các thay đổi đã thực hiện

### 🔹 Thay đổi gameplay
- Tốc độ chạy: **10 → 25**
- Tốc độ bắn: **0.2 → 0.05**
- Máu TankyEnemy: **75 → 500**
- Tốc độ Enemy: **5 → 12**

### 🔹 Thay đổi vật lý
- Lực nhảy: **12 → 24**
- Trọng lực: **0.5 → 3**

## 📚 Kiến thức đã học được
- Hiểu cấu trúc dự án Unity: `Assets`, `Scripts`, `Prefabs`, `Scenes`
- Biết cách đọc và phân tích code game thực tế
- Hiểu các design pattern:
  - Singleton
  - Observer
  - Inheritance
- Biết chỉnh sửa gameplay qua **Inspector**
- Làm quen với Git & GitHub:
  - Clone project
  - Commit & Push

3. Chọn thư mục project
4. Nhấn **Play** để chạy game
